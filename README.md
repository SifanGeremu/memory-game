React Memory Game
based in  fruit cards

A simple and fun Memory Game built with React. Test your memory by matching pairs of fruits before time runs out!

Features

Flip cards to find matching pairs

Track score and number of attempts

Responsive design for desktop and mobile

Built with React functional components and hooks



Installation

Clone the repository:

git clone https://github.com/SifanGeremu/memory-game

Navigate to the project directory:

cd react-memory-game

Install dependencies:

npm install

Start the development server:

npm start

The app will run at http://localhost:3000.

How to Play

Click on a card to reveal it.

Try to match it with its pair.

Continue until all pairs are matched.

Track your attempts and try to beat your best score!
